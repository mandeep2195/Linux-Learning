8. Click next

10. Click next
11. Click next
12. Slect Cores per processors as 6
13. Click next
14. Enter 4096 in memory for this virtual machine
15. Click next
16. Select Use network address translation (NAT)
17. Select Use LSI logic
18. Select SCSI

20. select the default value and click next
21. Click next
22. Click Finish
